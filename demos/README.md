## ğŸ’¡ Interview Tips

### Common Questions to Prepare

**HTML:**
- "Build a semantic blog post layout"
- "Create an accessible form with validation"
- "Implement responsive images"
- "Explain when to use `<article>` vs `<section>`"

**CSS:**
- "Center a div (multiple methods)"
- "Create a responsive grid without media queries"
- "Build a sticky footer"
- "Optimize CSS for performance"

### Live Coding Tips

1. **Think aloud**: Explain your approach
2. **Start simple**: Build incrementally
3. **Ask questions**: Clarify requirements
4. **Consider edge cases**: Mobile, accessibility, performance
5. **Test as you go**: Open in browser frequently
